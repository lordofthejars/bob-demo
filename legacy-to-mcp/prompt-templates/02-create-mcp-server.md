


# Creates an MCP Server

**Mode:** Quarkus Developer

**Context:**

Create an MCP Server using Quarkus that exposes the REST client created earlier as a tool. 

**Prompt:**

```
Create an MCP Server that provides tooling for the rest client interface.
```

**Result:**

* Bob uses the Quarkus MCP Server extension to create an MCP Server that exposes the REST endpoints to connect to the legacy app as MCP tools.

**Follow-up Actions:**

- Modifying the current Kubernetes deployment file to add the MCP server as a sidecar container
